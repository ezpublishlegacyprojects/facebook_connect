
 0. Activate extension
******************************

Activate the facebook_connect extension by using the admin interface
or in settings/override/site.ini.append(.php):

[ExtensionSettings]
ActiveExtensions[]=facebook_connect


 1. Clear caches
******************************

You will probably need to clear the INI and template caches.


 2. Setup facebook conenct
******************************

See Readme.txt on how to setup the facebook connect stuff.
